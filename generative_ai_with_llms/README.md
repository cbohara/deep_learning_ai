# [Generative AI with Large Language Models](https://learn.deeplearning.ai/courses/generative-ai-with-llms)
## Intro
How models are built 
- Find statistical patterns in massive datasets originally generated by humans   
- Train on millions of words with lots of compute   

Base models
- think of model's memory
- the more parameters, the more memory, and the more complex tasks it can perform  
- Using base models as is, or fine tune for specific use case, can use for specific use case without building from scratch   

LLMs
- take natural language and perform tasks like a human would
- prompt = text passed to LLM
- context window = space / memory available 
- completion = output of the model

Before transformers
- Used RNNs
- Required a lot of compute
- Predict next word just based on previous few words in the sentence

Transformers 
- Attention Is All You Need - from Google
- Scale efficiently for multi core CPUs
- Paralell processing
- Learn relevance of all the words to each other, not just the next word
- Attention weights are learned during LLM training
- Ex: word book strongly correlated with student and teacher 
- Self-attention = significantly improve ability to encode language
- Encoder and decoder work in parallel

Tokenization
- since models are just big statistical calculators
- convert words to numbers   
- each word (or parts of words) in a dictionary has a token ID
- need to use the same tokenizer to embed the input from words to numbers, and then numbers back to words

Embedding layer
- Trainable vector embedding space
- Each token is mapped to a vector
- Map words to a 3D space and see the distance between words

Positional encoding 
- Preserve info about word order from prompt 

Both embedding layer + positional encoding passed to self-attention layer >
- Analyze relationships of tokens / words
- Reflect importance of input words to all other words in the input sequence 
- Multi-headed self-attention = multiple learned in parallel (12 - 100 are common)
- Each self-attention head learns a different aspect of language
- Don't dictate ahead of time what the model will focus on

Attention weights applied to input data > feed forward network > vector of logits > final softmax layer where normalized into probabilty word for each score > output
